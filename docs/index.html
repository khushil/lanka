Get Started in Minutes

From installation to your first intelligent application

1

Install Prerequisites

# Install Node.js 20+
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install Neo4j
wget -O neo4j.deb https://neo4j.com/download/
sudo dpkg -i neo4j.deb
2

Clone & Configure

# Clone repository
git clone https://github.com/khushil/lanka.git
cd lanka

# Install dependencies
npm install

# Configure environment
cp .env.example .env
nano .env  # Add your Neo4j credentials
3

Initialize & Run

# Initialize database
npm run db:init

# Start development server
npm run dev

# Access at http://localhost:3000
